# AuMiau PetShop - Site Demo

Projeto front-end de um ecommerce pet desenvolvido com HTML, CSS e JavaScript puros.

## Objetivo

Simular uma loja virtual de pet shop com:
- catalogo de servicos
- catalogo de produtos por categoria
- carrinho de compras funcional
- simulacao de checkout

## Funcionalidades

- Layout responsivo para desktop e mobile
- Visual claro e divertido com animacoes
- Filtro de produtos por categoria:
  - caes
  - gatos
  - aves
  - roedores
- Carrinho com:
  - adicionar item
  - remover item
  - aumentar ou diminuir quantidade
  - calculo automatico do total
- Simulacao de compra com formulario

## Estrutura do Projeto

- index.html: estrutura da pagina
- style.css: estilos e responsividade
- script.js: dados, renderizacao, carrinho e checkout
- Leia-me.txt: requisitos iniciais

## Como Executar

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo index.html no navegador.

Opcao recomendada:
- usar a extensao Live Server para recarregamento automatico.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)

## Observacoes

- Este projeto e demonstrativo e nao possui backend.
- O checkout e uma simulacao sem processamento real de pagamento.
- As imagens usam links publicos.
